# BioSense+ Platform

A **BioSense+** é uma plataforma integrada composta por uma aplicação web e um sistema embarcado, projetada para aquisição, visualização e análise em tempo real de dados de pressão plantar e palmar. A solução permite a geração de mapas de calor e integração com infraestrutura de servidores para predições de diagnósticos médicos.

---

## Visão Geral

A plataforma BioSense+ realiza:

- Comunicação com dispositivo embarcado via **interface serial**
- Aquisição de dados de sensores de pressão
- Processamento e renderização de **mapas de calor em tempo real**
- Envio de dados para backend remoto
- Recebimento de **predições de diagnóstico médico** baseadas em modelos computacionais
